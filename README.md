#Grunt bootstrap

Just some playing around with them modern internet stuffs.


##Preparation

Install [node](http://nodejs.org/).

Then [bower](http://bower.io/)

    npm install -g bower
    
And also [grunt](http://gruntjs.com/)

    npm install -g grunt-cli



    
    


##Install

First run

    npm install
    
Then
 
    bower install
    
And finally
    
    grunt
